Quick Start

Add Traffic Cop bot detection to your website in under 2 minutes:

Method 1: Auto-initialization
<script src="https://traffic-cop-apii.vercel.app/traffic-cop-sdk.js" data-traffic-cop-key="YOUR_API_KEY"></script>
Method 2: Manual initialization
<script src="https://traffic-cop-apii.vercel.app/traffic-cop-sdk.js"></script>
<script>
  const trafficCop = TrafficCop.init('YOUR_API_KEY', {
    debug: true,
    onResult: function(result) {
      console.log('Bot detection result:', result);
    }
  });
</script>

Configuration Options

  • autoAnalyze: Automatically analyze visitors (default: true)
  • trackMouse: Track mouse movements (default: true)
  • trackBehavior: Track user behavior (default: true)
  • debug: Enable debug logging (default: false)
  • onResult: Callback function for results
